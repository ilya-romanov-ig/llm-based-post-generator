# LLM-based post generator

## Описание

Прикладной генератор постов на базе LLM (Qwen2.5:7b-instruct).

## Запуск

- Загрузка зависмостей: ```pip install -r requirements.txt```
- Загрузка модели ```ollama pull qwen2.5:7b-instruct```
- Запуск streamlit приложения ```streamlit run src/main.py```
